# Git Practice

This repository is a simple practice project created to learn and reinforce Git and GitHub fundamentals.

## Purpose

The goal of this repository is to:
- Practice creating a Git Repository
- Make commits locally
- Push changes to github using SSH
- Build confidence using Git as part of a dialy devemopment workflow

## What I learned

- A Git repository is just a folder Git tracks
- Commits are snapshots of changes over time
- GitHub stores a remote copy of my repository
- SSH allows secure authentication without passwords
